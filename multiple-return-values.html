Go by Example 中文版: 多返回值

Go 原生支持 _多返回值_。 这个特性在 Go 语言中经常用到，例如用来同时返回一个函数的结果和错误信息。

 
package main

import "fmt"

(int, int) 在这个函数中标志着这个函数返回 2 个 int。

 
func vals() (int, int) {
    return 3, 7
}

func main() {

这里我们通过 多赋值 操作来使用这两个不同的返回值。

 
    a, b := vals()
    fmt.Println(a)
    fmt.Println(b)

如果你仅仅需要返回值的一部分的话，你可以使用空白标识符 _。

 
    _, c := vals()
    fmt.Println(c)
}

$ go run multiple-return-values.go
3
7
7
